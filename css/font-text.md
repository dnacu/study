## text-align

텍스트 수평 정렬

> inline 레벨 요소에 text-align을 지정해도 적용되지 않는다. 왜냐하면 content의 너비가 width가 되어 중앙의 개념이 존재하지 않기 때문이다. 만약 중앙 정렬이 꼭 필요하다면 `display: block;` 혹은 감싸는 방법을 생각하자.
